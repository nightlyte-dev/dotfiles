## about:config

### Register Obsidian URI (Linux)

New entry -> Type = Boolean -> Value = `false`
```txt
network.protocol-handler.expose.obsidian
```

### Fullscreen in Window only

```
full-screen-api.ignore-widgets
```
- Value -> `true`

### Scroll through tabs

```
toolkit.tabbox.switchByScrolling
```
- Value -> `true`
